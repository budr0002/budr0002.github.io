QA for LIB2024

  1. Qus:What are the three components of Web?

    Ans:The components of Web are HTML, CSS, and Javascript.

  2. Qus:Define version control and give one example.

    Ans:Version control is a method of tracking changes to files over time so that they can be tracked and older versions can be used or referred to again. An example of version control is Github.

  3. Qus:What tag is used to create the title of a web page? Give an example.

    Ans:The <title> tag is used to create the title of a web page.
    An example is <title>This is the Title>/title>

  4. Qus:What tag is used to create a paragraph in html? Give an example.

    Ans:The p tag is used to create a paragraph.
    For example: <p>Type your paragraph content here.</p>
