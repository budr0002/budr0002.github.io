QA for LIB2024

  1. Qus:What are the three components of Web?

    Ans:The components of Web are HTML, CSS, and Javascript.

  2. Qus:Define version control and give one example.

    Ans:Version control is a method of tracking changes to files over time so that they can be tracked and older versions can be used or referred to again. An example of version control is Github.

  3. Qus:What tag is used to create the title of a web page? Give an example.

    Ans:The <title> tag is used to create the title of a web page.
    An example is <title>This is My Website</title>

  4. Qus:What tag is used to create a paragraph in html? Give an example.

    Ans:The p tag is used to create a paragraph.
    For example: <p>Type your paragraph content here.</p>

  5. Qus:Identify the odd one from these tags: <span>, <p>, <a>, <i> and state the reason?

    Ans:The <span> tag is the one that does not belong, because it is not a semantic element, it is a sectioning element.

  6. Qus:What are the tags required to create a navigation list in HTML? Give one example with two navigation links (https://algonquincollege.com, https://google.com

    Ans:A navigation list is created using the <nav> tag, and inside that would be the anchor tag <a> with the link as an href attribute. The anchor tags would be wrapped inside an unordered list using the <ul> tag, with list items for each link using the <li> tag.

    Example:

    • Algonquin College
    • Google
