QA for LIB2024

  1. Qus:What are the three components of Web?

    Ans:The components of Web are HTML, CSS, and Javascript.

  2. Qus:Define version control and give one example.

    Ans:Version control is a method of tracking changes to files over time so that they can be tracked and older versions can be used or referred to again. An example of version control is Github.

  3. Qus:What tag is used to create the title of a web page? Give an example.

    Ans:The <title> tag is used to create the title of a web page.
    An example is <title>This is My Website</title>

  4. Qus:What tag is used to create a paragraph in html? Give an example.

    Ans:The p tag is used to create a paragraph.
    For example: <p>Type your paragraph content here.</p>

  5. Qus:Identify the odd one from these tags: <span>, <p>, <a>, <i> and state the reason?

    Ans:The <span> tag is the one that does not belong, because it is not a semantic element, it is a sectioning element.

  6. Qus:What are the tags required to create a navigation list in HTML? Give one example with two navigation links (https://algonquincollege.com, https://google.com

    Ans:A navigation list is created using the <nav> tag, and inside that would be the anchor tag <a> with the link as an href attribute. The anchor tags would be wrapped inside an unordered list using the <ul> tag, with list items for each link using the <li> tag.

    Example:

    • Algonquin College
    • Google

  7. Qus:What element is used to create images? Name its attributes.

    Ans:The <img> element is used to create images. The attributes are src (tells the browser the source for the file), alt (tells the browser the alternative text), width and height (for the size of the image in pixels).

  8. Qus:Identify the odd one from these tags: <table>, <tr>, <b>, <td> and state the reason

    Ans:The odd one is <b> because this is the only tag in the list that is not related to tables. The <b> tag will render the text in bold.

  9. Qus:Give an example of color picker input.

    Ans:An example of a color picker input is: <input type="color" name="co" id="co">

  10. Qus:What is the use of name and value attributes? Give two examples.

    Ans:The name attribute is required to be used with the input tag, and is used as a reference for the data when it is submitted to the server. The value attribute is the actual value that gets submitted to the server. Value attribute is used with <button>, <input>, and <option> elements. An example of name attribute is <input type="text" name="first-name" placeholder="John">. An example of value attribute is <option value="chcake">Chocolate Cake</option>.

  11. Qus:Give an example of inline style.

    Ans:Inline style is used to apply CSS to a specific element directly within the HTML document. An example is:
    <p style="color:purple; font-size:20px;">This is a paragraph.</p>

  12. Qus:How would you link an external style sheet? Give an example.

    Ans:An external style sheet would be linked from inside the <head> element of the HTML document, using the <link> tag.

    Example:
    <head>
       <link rel="stylesheet" type="text/css" href="css/style.css">
